**Pick A Flick V1.4**

Feature backup before TN edition work — new pool behavior, on-hold list, and cabinet art options since v1.3.

**Highlights**

- **Category movie rotation** — Per-category shuffled decks; no repeat until the full eligible set has been shown; wheel, mystery picks, shuffle defaults, and manual placements all consume the pool.
- **On-Hold movies** — Pause unwatched titles from spin / coin-toss / shuffle pools; mutually exclusive with watched; `app/on-hold-movies.js`, result-card button, ON-HOLD FLICKS panel, bulk import.
- **Wheel art / TN backgrounds** — WHEEL ART panel swaps cabinet background (Arcade, TN, TN GPT, Mario, Star Wars, Lego variants); wheel disc, ticker, and spin button stay as live overlays.
- **Keeper + catalog** — Keeper adds sync to the result card; expanded catalog; arcade layout scaling.
- **Docs and tooling** — Git workflow docs/rules, FOCUS updates, OMDb normalize pass scripts and API-key helpers, mode-selector asset processors.

**After upgrading**

- Link or restore `app/on-hold-movies.js` if you use the on-hold list (same file-handle pattern as watched).
- Cabinet backgrounds live under `graphics/ui/*-background.png` (kebab-case names).
